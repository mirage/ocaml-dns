The type of a DNS resolver.

  1. | `Dnssec

val create : ?record_clients:bool -> ?cache_size:int -> ?ip_protocol:[ `Both | `Ipv4_only | `Ipv6_only ] -> feature list -> int64 -> (int -> string) -> Dns_server.Primary.s -> t

create ~record_clients ~cache_size ~ip_protocol features now rng primary creates the value of a resolver, pre-filled with root NS and their IP addresses. If ip_protocol is provided, and set to `V4_only, only IPv4 packets will be emitted. If `V6_only is set, only IPv6 packets will be emitted. If `Both (the default), either IPv4 and IPv6 packets are emitted. If record_clients is true (the default), the metrics of the resolver will include the amount of clients. This keeps a set of Ipaddr.t of all clients around, which may use some memory if it is a public resolver.

Some features can be specified, whether DNSSec validation should be done, whether query name minimisation should be done, and whether opportunistic encryption using TLS to the authoritative should be done.
